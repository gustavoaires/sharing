Task

Cancel

OAuth2 scope:

https://www.googleapis.com/auth/tasks
https://www.googleapis.com/auth/tasks.readonly

Code example:

// Get a task
Task task = google
   .taskOperations()
   .getTask("@default", "abcd1234");

// Save a task
Task task = google
   .taskOperations()
   .saveTask(
      "@default", 
      new Task(
         "Do a thing", 
         "Do it soon", 
         new Date()));

// Delete a task
google
   .taskOperations()
   .deleteTask("@default", "abcd1234");
