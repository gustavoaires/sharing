Task List

Cancel

OAuth2 scope:

https://www.googleapis.com/auth/tasks
https://www.googleapis.com/auth/tasks.readonly

Code example:

// Get a task list
TaskList taskList = google
   .taskOperations()
   .getTaskList("abcd1234");

// Save a task list
TaskList taskList = google
   .taskOperations()
   .saveTaskList(new TaskList("Stuff to do"));

// Delete a task list
google
   .taskOperations()
   .deleteTaskList("abcd1234");
